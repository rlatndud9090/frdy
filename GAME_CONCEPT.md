# 게임 컨셉: 마왕의 은밀한 조력

## 핵심 스토리

**주인공**: 마왕 (환생한 일반인)

**배경 설정**:
- 평범한 일반인이 마왕으로 환생함
- 원래 세계로 돌아가기 위한 유일한 방법: 용사의 손에 죽는 것
- 문제: 부하들이 충성심이 너무 강해서 용사를 막으려고 함

**게임 목표**:
- 부하들 **몰래** 용사를 도와야 함
- 용사가 마왕성을 성공적으로 돌파하도록 만들기
- 들키지 않으면서 용사의 승리를 보장해야 함

## 게임 메커니즘

### 1. 맵 구조 (절차적 생성)

**마왕성의 혼돈 마법**:
- 문을 열 때마다 구조가 뒤바뀌는 혼돈의 마법이 걸려있음
- 로그라이크 특성을 위한 절차적 생성 구현

**맵 구조**:
- 슬레이 더 스파이어 스타일의 **Node-Edge Graph** 구조
- 여러 **'층'**으로 구성 (슬더스의 '막'과 동일한 역할)
- 각 층은 연결된 노드들로 구성됨

**Node 타입**:
- 🗡️ **전투 노드**: 적과의 전투 발생
- ❓ **이벤트 노드**: 특수 이벤트 발생

### 2. 전투 시스템

**기본 구조**:
- 턴제 전투
- 용사와 적 각각 다음 속성 보유:
  - 공격력
  - 체력
  - 행동 패턴

**용사 성장**:
- 전투 승리 시 강화
- 이벤트를 통한 강화
- 공격력, 체력, 행동 패턴이 점진적으로 향상

### 3. 마왕의 개입 시스템

**행동 카드 사용**:
- 매 턴마다 **'마력'**을 소모하여 **'행동 카드'** 사용
- 행동 카드로 용사를 **조력** 또는 **방해** 가능
- 전략적 선택이 핵심

**마왕의 성장**:
- 용사가 전투를 승리할 때마다 새로운 **행동 카드** 획득
- 환생 컨셉: 처음엔 약한 힘만 사용 가능 → 점점 깨달아 강해짐

### 4. 의심 시스템 (핵심 메커니즘)

**의심 수치**:
- 용사를 **도우면** → 의심 수치 **증가** ⬆️
- 용사를 **방해하면** → 의심 수치 **감소** ⬇️
- 의심 수치가 **MAX** 도달 → 부하들의 반란 → **게임 오버**

**전략적 중요성**:
- 너무 노골적으로 도우면 들킴
- 적절히 방해하면서 은밀하게 조력해야 함
- 의심받지 않는 균형 유지가 승리의 열쇠

### 5. 이벤트 및 진로 조작

**이벤트 발생 시**:
- 마왕의 선택지 제공 (조력/방해)
- 선택에 따라 의심 수치 변동

**진로 수정**:
- 용사가 갈래길에서 방향 선택 시
- 마왕이 용사의 **진로를 은밀히 수정** 가능
- 진로 수정도 의심 수치에 영향
